# HarmonyOS 互动卡片工程文件描述

## 工程概述

本工程是一个 **HarmonyOS 互动卡片（Live Form）演示项目**，展示了 HarmonyOS 互动卡片能力的完整实现。项目包含睡眠、快递、运动、音乐四种不同场景的互动卡片示例，每种卡片都具备独特的动画效果和交互体验。

## 互动卡片能力说明

### 四种卡片类型

1. **睡眠卡片**: 三叶草起床/入睡动画，展示睡眠健康数据
2. **快递卡片**: 陀螺仪驱动的憨憨移动动画
3. **运动卡片**: 卡路里燃烧动画，运动状态切换
4. **音乐卡片**: 专辑封面跟随动画帧同步移动特效

### 核心通信机制

| Action 类型 | 用途 | 目标 |
|------------|------|------|
| `CALL` | 播放/切歌/收藏操作 | UIAbility |
| `MESSAGE` | 请求激活互动卡片 | FormExtensionAbility |
| `ROUTER` | 跳转到应用落地页 | UIAbility |
