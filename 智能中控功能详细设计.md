## 🎯 功能定位与创新点

### 1.1 功能概述

智能中控互动卡片是一个**全屋智能家居控制中心**，将传统的多个独立APP控制功能整合到一个HarmonyOS互动卡片中。用户无需打开APP，在桌面即可通过**丰富的手势交互**完成全屋智能设备控制。

### 1.2 核心创新点（与普通天气卡片的本质区别）

| 对比维度 | 普通天气卡片 | 智能中控卡片 | 优势 |
|---------|------------|------------|------|
| **设备数量** | 1种数据展示 | **8种设备 + 5种场景** | ×8倍复杂度 |
| **交互方式** | 简单点击 | **点击/滑动/长按/双击/双指缩放** | 5种手势 |
| **状态数量** | 4种天气状态 | **32种设备状态组合** | ×8倍状态数 |
| **数据可视化** | 简单文本 | **实时曲线图 + 环形进度图 + 能耗热力图** | 3种图表 |
| **联动逻辑** | 无 | **场景联动 + 条件自动化** | 复杂业务逻辑 |
| **持久化存储** | 无 | **设备状态持久化 + 历史数据记录** | 数据层完整 |
| **代码规模** | ~500行 | **~3000行** | ×6倍代码量 |

## 📱 详细功能设计

### 3.1 支持的智能设备（8种）

#### 3.1.1 智能灯光控制
- **开关控制**: 点击开关图标切换
- **亮度调节**: 横向滑动0-100%无级调节
- **色温调节**: 纵向滑动暖光-冷光2700K-6500K
- **颜色选择**: 长按弹出RGB调色板（1600万色）
- **分组控制**: 客厅/卧室/书房分组开关

#### 3.1.2 智能空调控制
- **开关控制**: 电源开关
- **温度调节**: 16°C-30°C 滑动调节
- **模式切换**: 制冷/制热/送风/除湿/自动
- **风速调节**: 低/中/高/自动
- **风向控制**: 上下扫风/左右扫风

#### 3.1.3 智能窗帘控制
- **开合控制**: 0-100% 精确位置
- **开合速度**: 慢速/中速/快速
- **定时开合**: 日出日落自动触发

#### 3.1.4 安防系统控制
- **布防模式**: 在家/离家/睡眠布防
- **报警状态**: 实时显示门磁、人体感应
- **摄像头**: 点击查看实时画面缩略图
- **门锁状态**: 门锁开关、临时密码

#### 3.1.5 空气净化器
- **开关控制**: 电源开关
- **风速调节**: 自动/低速/中速/高速
- **空气质量**: PM2.5实时数值 + 颜色指示
- **滤芯寿命**: 进度条显示剩余寿命

#### 3.1.6 智能插座
- **开关控制**: 每个插座独立开关
- **功率监测**: 实时功率显示
- **定时开关**: 倒计时功能
- **过载保护**: 功率超标自动断电

#### 3.1.7 环境传感器
- **温度**: 室内温度实时显示
- **湿度**: 相对湿度百分比
- **PM2.5**: 空气质量指数
- **CO2**: 二氧化碳浓度

#### 3.1.8 能耗监测
- **实时功率**: 全屋总功率
- **今日用电**: kWh 统计
- **本月电费**: 预估费用
- **设备排行**: 能耗TOP3设备

### 3.3 自动化规则引擎

#### 3.3.1 支持的触发条件
1. **时间触发**: 每天/每周/指定时间点
2. **设备状态触发**: 温度>28°C → 开空调
3. **日出日落触发**: 日落 → 开窗帘
4. **定位触发**: 离家1km → 离家模式
5. **设备联动触发**: 开门 → 开灯

#### 3.3.2 规则示例
```
规则1: 温度自动化
IF 室内温度 > 28°C AND 有人在家
THEN 开启空调 → 制冷模式 → 26°C

规则2: 日落自动化
IF 日落时间到达
THEN 关闭窗帘 + 开启客厅主灯

规则3: 离家自动化
IF 所有家庭成员离家 > 5分钟
THEN 执行离家模式
```

## 🔧 核心技术实现详解

### 4.1 设备状态机设计

```typescript
/**
 * 设备状态枚举
 */
export enum DeviceType {
  LIGHT = 'light',
  AIR_CONDITIONER = 'ac',
  CURTAIN = 'curtain',
  SECURITY = 'security',
  PURIFIER = 'purifier',
  SOCKET = 'socket',
  SENSOR = 'sensor',
  CAMERA = 'camera'
}

/**
 * 设备状态接口
 */
export interface IDeviceState {
  id: string;
  name: string;
  room: string;
  type: DeviceType;
  isOn: boolean;
  value: number;           // 亮度/温度/开合度
  mode?: number;           // 工作模式
  lastUpdate: number;
}

/**
 * 状态机核心类
 */
export class DeviceStateMachine {
  private states: Map<string, IDeviceState> = new Map();
  private listeners: Array<(device: IDeviceState) => void> = [];

  /**
   * 更新设备状态
   */
  updateState(deviceId: string, updates: Partial<IDeviceState>): void {
    const device = this.states.get(deviceId);
    if (device) {
      Object.assign(device, updates);
      device.lastUpdate = Date.now();
      this.notifyListeners(device);
      this.checkAutomationRules(device);
    }
  }

  /**
   * 检查自动化规则
   */
  private checkAutomationRules(device: IDeviceState): void {
    // 遍历所有规则，检查触发条件
    AutomationEngine.getInstance().evaluateRules(device);
  }
}
```

### 4.2 手势识别引擎

```typescript
/**
 * 手势识别器
 */
export class GestureRecognizer {
  private touchStartX: number = 0;
  private touchStartY: number = 0;
  private touchStartTime: number = 0;
  private touchCount: number = 0;

  /**
   * 识别手势类型
   */
  recognizeGesture(event: TouchEvent): GestureType {
    const deltaTime = Date.now() - this.touchStartTime;
    const deltaX = event.touches[0].x - this.touchStartX;
    const deltaY = event.touches[0].y - this.touchStartY;

    // 长按识别 (>500ms)
    if (deltaTime > 500 && Math.abs(deltaX) < 10 && Math.abs(deltaY) < 10) {
      return GestureType.LONG_PRESS;
    }

    // 滑动识别 (>50px)
    if (Math.abs(deltaX) > 50) {
      return deltaX > 0 ? GestureType.SWIPE_RIGHT : GestureType.SWIPE_LEFT;
    }

    return GestureType.CLICK;
  }
}
```

   * 绘制温度湿度曲线
