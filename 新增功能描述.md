# 新增功能描述文档 - 天气互动卡片

## 功能概述

在原有4种互动卡片（睡眠、快递、运动、音乐）基础上，**新增第5种互动卡片：天气互动卡片（Weather Live Card）**。

天气互动卡片是一个功能强大的实景天气展示卡片，具备动态天气粒子动画、实时气象数据展示、城市切换等核心功能，大幅提升了项目的完整性和演示效果。

---

## ✨ 新增功能亮点

### 1. 四种天气场景动画

| 天气类型 | 动画效果 | 触发方式 |
|---------|---------|---------|
| ☀️ **晴天** | 太阳旋转光晕 + 漂浮云朵动画 | 点击太阳图标 |
| ⛅ **多云** | 云层缓慢移动 + 光影变化 | 点击云朵图标 |
| 🌧️ **雨天** | 雨滴下落粒子系统 + 水波纹 | 点击雨滴图标 |
| ❄️ **雪天** | 雪花飘落动画 + 积雪效果 | 点击雪花图标 |

### 2. 实时气象数据展示

- **温度显示**: 当前温度 + 最高/最低温
- **湿度显示**: 相对湿度百分比
- **风力显示**: 风向 + 风力等级
- **空气质量**: AQI指数 + 等级提示
- **更新时间**: 数据最后更新时间戳

### 3. 城市切换功能

- 支持5个热门城市快速切换
- 北京、上海、广州、深圳、成都
- 城市切换带动画过渡效果
- 每个城市对应不同的默认天气

### 4. 互动卡片溢出效果

点击天气卡片主区域，激活**互动卡片溢出动画**：
- 全屏天气动画展示
- 详细的7天天气预报
- 小时级温度趋势图
- 生活指数建议（穿衣、紫外线、运动等）

## 🔧 核心技术实现

### 1. 粒子动画系统

```typescript
// 雨滴粒子类
class Raindrop {
  x: number = Math.random() * 400
  y: number = Math.random() * 800
  speed: number = 3 + Math.random() * 5
  length: number = 10 + Math.random() * 20
  
  update(): void {
    this.y += this.speed
    if (this.y > 800) {
      this.y = -20
      this.x = Math.random() * 400
    }
  }
}

// Canvas渲染雨滴
@Builder
RainParticleSystem() {
  Canvas(this.rainCanvas)
    .width('100%')
    .height('100%')
    .onReady(() => {
      this.startRainAnimation()
    })
}
```

### 2. 天气状态机

```typescript
enum WeatherType {
  SUNNY = 'sunny',
  CLOUDY = 'cloudy',
  RAINY = 'rainy',
  SNOWY = 'snowy'
}

class WeatherStateMachine {
  currentState: WeatherType = WeatherType.SUNNY
  
  transitionTo(newState: WeatherType): void {
    // 状态切换带动画过渡
    this.playTransitionAnimation(this.currentState, newState)
    this.currentState = newState
  }
}
```

### 3. 卡片通信机制

```typescript
// 动态卡片 → 应用：切换城市
postCardAction(this, {
  action: 'CALL',
  abilityName: ENTRY_ABILITY,
  params: {
    method: 'cardAction',
    actionType: 'CHANGE_CITY',
    cityId: cityId
  }
})

// 应用 → 互动卡片：更新天气数据
WeatherFileStore.storeWeatherData(context, weatherData)
```

## 🎬 运行效果说明

### 桌面卡片效果

1. **默认展示**: 2×4 尺寸天气卡片，显示当前城市、温度、天气图标
2. **点击图标**: 切换天气类型，播放对应动画
3. **点击城市名**: 弹出城市选择器，快速切换城市
4. **点击主区域**: 激活互动卡片溢出，进入全屏天气动画

### 互动卡片效果

1. **全屏动画**: 沉浸式天气粒子动画覆盖全屏
2. **数据面板**: 右侧滑出详细气象数据面板
3. **7天预报**: 横向滚动展示一周天气预报
4. **生活指数**: 底部展示各类生活建议图标

## 🔒 新增权限

在 `module.json5` 中新增权限声明：

```json
{
  "name": "ohos.permission.INTERNET",
  "reason": "$string:internet_permission_reason"
}
```

**用途**: 用于获取实时天气数据（模拟网络请求）
